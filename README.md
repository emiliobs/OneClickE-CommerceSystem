# 🛒 OneClick E-Commerce System

Welcome to **OneClick**, a modern, full-stack e-commerce solution built with **.NET 8** and **Blazor WebAssembly**. This platform provides a seamless shopping experience with a robust, secure, and scalable backend architecture.

---

## ✨ Key Features
* **Responsive UI:** Modern design that works beautifully on desktop and mobile.
* **Shopping Cart & Checkout:** Intuitive, dynamic cart management and secure order placement.
* **Admin Dashboard:** Comprehensive inventory and category management for store administrators.
* **RESTful API:** Fully documented OpenAPI (Swagger) backend.
* **N-Layer Architecture:** Clean separation of concerns (Core, Infrastructure, WebUI).
